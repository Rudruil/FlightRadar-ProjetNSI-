A propos de notre projet


Notre projet a pour objectif de créer une application qui permet d’afficher une carte en tant réel de Chelles et de son trafic aérien. De plus cette application permet de connaître le numéros de vol, et toutes les informations concernant le vol ces dernières contenues dans une base de donnée, créée par le groupe.

Le groupe est composé de 3 personnes entre lesquelles le travail est équitablement répartie. En effet,

L’élève 1 doit :
-coder le module logiciel pour décoder le signal radio pour le protocole ADSB
-coder le module logiciel pour stocker les information dans la base de donnée
-coder le module logiciel pour stocker dans une IHM la carte en temps réel de la position des avions

L’élève 2 doit :
-coder le module logiciel pour décoder le signal radio ACARS
-coder le module logiciel pour stocker les condition climatique dans la base de donnée
-coder le module logiciel pour afficher dans l’IHM la trajectoire des avions

L’élève 3 doit :
-installer et configurer le serveur web et la base de donné
-coder le module logiciel pour calculer la distance entre 2 avions
-coder le module logiciel pour produire une alerte sonore et visuel en cas de rapprochement de 2 avions
-coder le site web pour afficher sur la carte en temps réel la position des différents avions et des conditions de vol


durant le développement de ce projet, certaines idées de développement nous sont venues à l’esprit.
Tout d’abord, de ne pas se limiter à Chelles et d’étendre le logiciel à la France et pourquoi pas au monde entier. L’objectif serait à terme de pouvoir choisir le vol dont on veut les informations peut importes où il se trouve dans le monde.
